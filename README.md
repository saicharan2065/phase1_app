# Phase 1 App

This project fulfills the Phase 1 requirements using Python and Gradio.

## Features
- **Data Sources:** Load Hugging Face datasets, upload CSV/JSON/XLSX, import URLs, and generate synthetic data.
- **Reference Validation:** Source and Reference dataset validation, schema matching, and mismatch tables.
- **Model Management:** System vs User models, Hugging Face installation, activation, deletion, and system resource (RAM/Disk/VRAM) display.

## Installation
```bash
pip install -r requirements.txt
```

## Running the App
```bash
python app.py
```
